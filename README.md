# 🎬 VidRAG – Video RAG System

## 🚀 Overview
Distributed AI system for video understanding using RAG.

## ✨ Features
- YouTube ingestion
- Whisper transcription
- Hybrid search (FAISS + BM25)
- LLM QA
- Timestamp navigation
